GTK+ with CSS

Starting with version 3, GTK uses CSS3 for styling and although it's a subset of the CSS3 standard so not everything is supported, you can use it to greatly improve the styling of your applications.

CSS

(Cascading Style Sheets), are used to style documents written in languages for web content creation such as HTML, XHTML, SVG, XUL, ect. The current version is CSS3 and is maintained by the W3C, (World Wide Web Consortium)
